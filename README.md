# 📰 IntelNews Engine

An end-to-end Natural Language Processing (NLP) project that analyzes news headlines using Machine Learning and Transformer-based models. The system classifies news articles, detects sentiment, predicts headline trustworthiness, rewrites headlines into multiple styles, and recommends similar news articles through an interactive interface.

---

##  Features

- News Category Classification (World, Sports, Business, Sci/Tech)
- Sentiment & Emotion Detection using Hugging Face Transformers
- Trust Score Prediction (0–100)
- Headline Rewriter
  - Professional
  - Formal
  - Clickbait
  - Social Media
- Similar News Recommendation Engine
- Interactive User Interface
- User Feedback Collection

1. Load and preprocess dataset
2. TF-IDF feature extraction
3. Category prediction using Logistic Regression
4. Emotion detection using DistilBERT
5. Trust score prediction
6. Headline rewriting
7. Similar news recommendation
8. Interactive headline analysis

---

## Key Functionalities

### News Category Prediction

Predicts the category of any news headline using TF-IDF vectorization and Logistic Regression.

### Emotion Detection

Detects Positive, Negative, or Neutral sentiment using a Hugging Face Transformer model.

### Trust Score Prediction

Assigns a trust score based on clickbait indicators, punctuation, capitalization, and sentiment.

### Headline Rewriter

Generates multiple versions of the same headline in different writing styles.

### Similar News Recommendation

Recommends similar news articles using cosine similarity.
